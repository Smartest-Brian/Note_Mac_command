#  macOS 內建的替代方案 (find)

---

### 1. 確認你正在使用的 Shell
在終端機輸入以下指令：
```bash
echo $SHELL
```
* 如果顯示 `/bin/zsh`，設定檔是 `~/.zshrc`。
* 如果顯示 `/bin/bash`，設定檔是 `~/.bash_profile` 或 `~/.bashrc`。

---

### 2. 加入 Alias 設定
請按照以下步驟操作：

1.  **打開設定檔**（以 Zsh 為例）：
    ```bash
    nano ~/.zshrc
    ```
2.  **將以下內容貼到檔案最後面**：
    為了避免單引號與雙引號衝突，建議寫法如下：
    ```bash
    alias tree="find . -print | sed -e 's;[^/]*/;|____;g;s;____|; |;g'"
    ```
3.  **儲存並離開**：
    按下 `Ctrl + O` 然後按 `Enter` 儲存，再按下 `Ctrl + X` 離開。

---

### 3. 讓設定立即生效
修改完設定檔後，目前的終端機視窗還不會發現新設定，請執行：
```bash
source ~/.zshrc
```
*(如果你是 Bash，請改用 `source ~/.bash_profile`)*

---

### 4. 測試成果
現在，你直接在任何資料夾輸入：
```bash
tree
```
它就會噴出你剛才設定的那串樹狀圖了！

---

### 進階建議：寫成 Function (更強大)
`alias` 有個缺點是無法帶入參數（例如你想指定特定路徑）。如果你希望輸入 `tree src` 就能看 `src` 資料夾，建議寫成 **Function**：

同樣在 `~/.zshrc` 裡面加入：
```bash
tree() {
    find "${1:-.}" -print | sed -e 's;[^/]*/;|____;g;s;____|; |;g'
}
```
* `${1:-.}` 的意思是：如果有給路徑就用該路徑，沒給就預設用目前目錄 `.`。
